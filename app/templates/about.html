Crypto Brella

Cipher Toolbox

CryptoBrella is a long-term personal project created by a3xgvo.


CryptoBrella is a collection of cryptography-related tools, documents, and challenges. It mainly deals with practical tasks such as analysis, decoding, and experimentation.


The site is primarily intended for cryptography decoders.


The name “CryptoBrella” comes from Splat Brella, a weapon from Splatoon that a3xgvo used frequently. The idea of grouping various cryptographic topics under a single umbrella came later.


Currently, the site includes:

  • 50+ cryptographic tools
  • Documentation for techniques and references
  • Original challenges
  • The Ingress Passcode Challenge

Many of the tools include small conveniences that reflect personal preferences. For example, most converters update automatically as you type without requiring an explicit action, and the interface is designed with mobile use in mind.


The site continues to be updated incrementally.
